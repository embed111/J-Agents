# 方法: 系统思维分析

- 最后更新: 2026-02-24

## 适用问题
1. 问题由多个环节相互影响，单点优化效果不稳定。
2. 结果存在时间延迟，短期观察容易误判。
3. 需要识别反馈回路和关键杠杆点。

## 核心步骤
1. 确定系统边界与参与要素。
2. 画出关键因果链与反馈回路（强化回路/平衡回路）。
3. 识别时滞与测量盲区。
4. 找出高杠杆干预点并评估副作用。
5. 设计分阶段干预与观测指标。

## 输出格式
1. 系统边界
2. 关键回路
3. 杠杆点
4. 时滞与风险
5. 干预方案与监测指标

## 常见误区
1. 只盯局部指标，忽略全局反馈。
2. 把相关性当成因果链。
3. 忽略时滞导致过早否定策略。
