# 方法: 第一性原理分析

- 最后更新: 2026-02-24

## 适用问题
1. 现有方案复杂但收益不清晰。
2. 团队默认“只能这样做”，但缺少底层证明。
3. 需要从零重建问题定义与方案空间。

## 核心步骤
1. 拆解目标: 将目标拆到不可再分的真实需求。
2. 区分事实与惯例: 标记哪些是客观约束，哪些是历史习惯。
3. 回到底层约束: 明确物理/经济/认知/制度边界。
4. 重构方案: 在约束内重新组合最小可行路径。
5. 成本收益校验: 比较新旧路径的边际成本和边际收益。

## 输出格式
1. 底层目标: 真正要优化的结果。
2. 不可违背约束: 绝对边界条件。
3. 可重构部分: 旧方案中可以被替代或删除的环节。
4. 最小可行解: 从底层重建后的第一版方案。

## 常见误区
1. 把“行业惯例”误当“自然规律”。
2. 过度抽象，导致方案无法落地。
3. 忽略迁移成本，纸面最优但实施失败。
