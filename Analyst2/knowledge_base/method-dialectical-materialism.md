# 方法: 唯物辩证法分析

- 最后更新: 2026-02-24

## 适用问题
1. 目标冲突明显（效率 vs 质量、速度 vs 稳定）。
2. 系统处于变化中，需要分阶段策略。
3. 需要识别“主次矛盾”并安排资源。

## 核心步骤
1. 定义系统与边界: 明确对象、参与方、约束与时间范围。
2. 识别矛盾: 列出关键冲突对，区分主矛盾与次矛盾。
3. 分析转化条件: 判断矛盾在何种条件下会相互转化或缓解。
4. 制定阶段策略: 给出当前阶段优先级与后续切换条件。
5. 验证与迭代: 用指标观察矛盾变化，按触发条件调整策略。

## 输出格式
1. 主矛盾: 当前最影响结果的冲突。
2. 次矛盾: 需要监控但暂不优先处理的冲突。
3. 阶段策略: 本阶段做什么、不做什么、何时切换。
4. 观测指标: 用于判断策略有效性的量化信号。

## 常见误区
1. 把所有矛盾都当主矛盾，导致资源分散。
2. 只给静态结论，不给阶段切换条件。
3. 混淆事实与价值判断，导致论证不稳定。
