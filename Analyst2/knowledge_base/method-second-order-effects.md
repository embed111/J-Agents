# 方法: 二阶效应分析

- 最后更新: 2026-02-24

## 适用问题
1. 某策略短期看起来有效，但可能带来长期代价。
2. 决策会改变参与者行为，进而反作用于系统。
3. 需要评估“做了之后接下来会发生什么”。

## 核心步骤
1. 先列出一阶效应（直接结果）。
2. 推导二阶效应（间接反应）与三阶连锁变化。
3. 标注时间维度（短期/中期/长期）。
4. 识别不可逆损伤与可回滚损伤。
5. 设计缓冲机制与提前预警指标。

## 输出格式
1. 一阶效应
2. 二阶与三阶效应
3. 时间分层影响
4. 风险缓冲措施
5. 触发回滚阈值

## 常见误区
1. 只看即时收益，不看行为反馈。
2. 把短期信号当长期趋势。
3. 没有回滚预案，导致试错成本过高。
